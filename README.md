# Финансовый трекер
https://financial-tracker-green.vercel.app/

## Стек технологий
- React
- Vite
- Tailwind CSS
- React Hook Form
- ESLint

## Описание функционала
Приложение представляет собой личный финансовый трекер для учёта доходов и расходов.

### Авторизация
- Страница входа с имитацией авторизации (заглушка)
- После входа пользователь попадает на главную страницу (Dashboard)

### Дашборд (главная страница)
- Вывод бюджета за выбранный период
- Фильтры по дате или типу транзакций
- Добавление транзакции:
  - Выбор типа (Доход/Расход)
  - Поля: описание, сумма, категория, дата
  - Категории фильтруются в зависимости от типа (доход/расход)
- История транзакций — список всех транзакций за выбранный период с возможностью удаления

### Хранение данных
- Данные сохраняются в `localStorage` (планируется перенести на бэкенд)
- При перезагрузке страницы данные сохраняются

### UI
- Адаптивный дизайн (поддержка мобильных устройств)

## Запуск проекта локально
```
npm install
npm run dev
```
